## Dove Studio

### Informazioni sull'Istituto

- **Nome:** Istituto Tecnico Tecnologico Statale "Silvano Fedi - Enrico Fermi"
- **Località:** Pistoia, Provincia di Pistoia, Italia
- **Sito Web:** [www.ittfedifermi.it](https://www.ittfedifermi.it/)

#### Descrizione dell'Istituto

L’I.T.T.S. “FEDI-FERMI” è un Istituto Tecnico Tecnologico che offre corsi di studio della durata di cinque anni. I primi due anni costituiscono un biennio comune, seguito da trienni (due anni più uno) per gli indirizzi di studi specializzati, tra cui Informatica e Telecomunicazioni. Al termine del corso di studi, l’Istituto rilascia il Diploma di Perito Tecnico Tecnologico.

#### Storia dell'Istituto

L’Istituto Tecnico Tecnologico Statale "Silvano Fedi - Enrico Fermi" ha assunto la sua attuale denominazione nell'anno scolastico 2011-2012, a seguito dell’accorpamento dell’Istituto Tecnico per Geometri Enrico Fermi da parte dell’Istituto Tecnico Industriale Silvano Fedi.

Enrico Fermi (1901-1954) è stato uno dei più grandi fisici italiani e mondiali, noto per i suoi studi pionieristici sulla Meccanica Quantistica e la fisica nucleare. Silvano Fedi, nativo di Pistoia, è considerato un martire della lotta antifascista e ha dato un contributo significativo alla resistenza partigiana durante la Seconda Guerra Mondiale.

Il percorso di studi in "Informatica" offre una formazione approfondita nel campo dei sistemi informatici, dell'elaborazione dell'informazione e dello sviluppo di applicazioni software.

I diplomati in "Informatica" acquisiscono competenze specifiche, tra cui:

- Analisi e progettazione di sistemi informatici.
- Sviluppo di software per applicazioni desktop, web e mobile.
- Gestione e manutenzione dei sistemi informatici e delle reti.
- Utilizzo avanzato di linguaggi di programmazione e tecnologie informatiche.

#### Articolazioni del Percorso

Il percorso si concentra sullo sviluppo di competenze nell'ambito dell'informatica, inclusi:

1. **Analisi e Sviluppo Software**: Approfondimento delle metodologie di analisi, progettazione e sviluppo di software per diverse piattaforme e applicazioni.
   
2. **Gestione dei Sistemi Informatici**: Studio della gestione e manutenzione dei sistemi informatici, inclusi server, reti e infrastrutture tecnologiche.

- Analizzare i requisiti software e progettare soluzioni efficaci.

- Collaborare in team multidisciplinari per la realizzazione di progetti software.

## Guida alla Configurazione del Mio Setup del Terminale Personalizzato

Questa guida fornisce istruzioni dettagliate per configurare il tuo terminale con il mio setup personalizzato, che include font, temi e funzionalità avanzate per ottimizzare la tua esperienza di sviluppo e lavoro.

Segui i passaggi descritti di seguito per creare un ambiente di lavoro personalizzato e efficiente.

### Prerequisiti

Prima di iniziare la configurazione del terminale, è essenziale verificare che tutti i prerequisiti siano soddisfatti. Assicurati di avere i seguenti elementi a disposizione sul tuo sistema:

- **Windows Terminal**: Si tratta di un emulatore di terminale moderno e altamente personalizzabile progettato per il sistema operativo Windows. Puoi ottenere Windows Terminal gratuitamente dal Microsoft Store o dal repository ufficiale su GitHub. Verifica di averlo installato correttamente sul tuo sistema.

- **Winget**: Questo è il gestore di pacchetti ufficiale di Windows, progettato per semplificare il processo di installazione e gestione del software su piattaforma Windows. Se non lo hai già fatto, puoi seguire le istruzioni di installazione disponibili sul sito Web ufficiale di Microsoft.

- **pwsh**: Questa è la shell di comando di PowerShell Core, una versione moderna e multipiattaforma di PowerShell. Verifica la sua installazione aprendo una finestra di PowerShell Core e digitando il comando `pwsh`, quindi premendo `Invio`. Se il prompt di PowerShell Core appare senza errori, significa che l'installazione è avvenuta correttamente.

Una volta confermato che tutti i prerequisiti siano soddisfatti, puoi procedere con la configurazione del terminale seguendo le istruzioni fornite nella sezione successiva del README.

### Installazione

#### 1. Configurazione di Scoop

Prima di procedere con la configurazione di Scoop, assicurati di seguire attentamente i passaggi seguenti:

1. **Impostazione della Politica di Esecuzione**:
   
    Per consentire l'esecuzione degli script, è necessario impostare la politica di esecuzione di PowerShell. Esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
    ```

    Questo comando imposta la politica di esecuzione su "RemoteSigned" per l'utente corrente, consentendo l'esecuzione di script remoti firmati da un editore attendibile.

2. **Installazione di Scoop**:

    Per installare Scoop, esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
    ```

    Questo comando scarica lo script di installazione di Scoop da Internet e lo esegue, consentendo l'installazione di Scoop sul tuo sistema.

Una volta completati entrambi i passaggi sopra descritti, Scoop sarà installato e configurato correttamente sul tuo sistema. Potrai quindi utilizzare Scoop per gestire i pacchetti e le applicazioni sul tuo sistema Windows.

#### 2. Installazione di Git

Per installare Git utilizzando Winget, esegui il seguente comando in una finestra di PowerShell:

```powershell
winget install --id Git.Git -e --source winget
```

Questo comando utilizzerà Winget per cercare il pacchetto Git nell'elenco dei pacchetti disponibili e lo installerà sul tuo sistema. Assicurati di avere una connessione Internet attiva durante l'esecuzione di questo comando in modo che Winget possa scaricare e installare Git correttamente.

Una volta completato il processo di installazione, Git sarà disponibile sul tuo sistema e potrai iniziare a utilizzarlo per gestire i tuoi repository e il controllo delle versioni. Assicurati di verificare l'installazione eseguendo `git --version` in una finestra di PowerShell per confermare che Git sia stato installato correttamente e per visualizzare la sua versione.

#### 3. Installazione dei Font Nerd

1. **Aggiunta del Bucket di nerd-fonts a Scoop**:

    Per aggiungere il bucket di nerd-fonts a Scoop, esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    scoop bucket add nerd-fonts
    ```

    Questo comando aggiungerà il bucket di nerd-fonts alla lista dei bucket gestiti da Scoop, consentendo così di accedere ai font Nerd disponibili per l'installazione.

2. **Installazione del Font JetBrainsMono Nerd Font**:

    Per installare il font JetBrainsMono Nerd Font, esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    scoop install JetBrainsMono-NF
    ```

  > **Nota:** Dopo aver eseguito questo comando, assicurati che il font sia presente nella cartella `%USERPROFILE%\scoop\apps\JetBrainsMono-NF\current`. Una volta trovati i file del font, selezionali tutti e fai clic con il tasto destro del mouse per selezionare l'opzione "Installa". Questo processo installerà il font sul tuo sistema, rendendolo disponibile per l'uso nel terminale e in altre applicazioni.

Una volta completati entrambi i passaggi sopra descritti, i font Nerd saranno installati sul tuo sistema e potrai utilizzarli per personalizzare l'aspetto del tuo terminale.

#### 4. Installazione di Extra Tools

1. **Aggiunta del Bucket extras a Scoop**:

    Per aggiungere il bucket extras a Scoop, esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    scoop bucket add extras
    ```

    Questo comando aggiungerà il bucket extras alla lista dei bucket gestiti da Scoop, consentendo di accedere ai pacchetti aggiuntivi disponibili per l'installazione.

2. **Installazione degli Strumenti Extra**:

    Per installare gli strumenti aggiuntivi, esegui i seguenti comandi uno dopo l'altro in una finestra di PowerShell:

    ```powershell
    scoop install extras/psreadline
    scoop install extras/terminal-icons
    scoop install extras/z
    ```

    Questi comandi installeranno rispettivamente i seguenti strumenti:
    
    - `psreadline`: Aggiunge funzionalità avanzate di editing della riga di comando a PowerShell.
    - `terminal-icons`: Fornisce icone aggiuntive per il terminale per migliorare la visualizzazione dei file e delle directory.
    - `z`: Implementa una navigazione veloce della directory basata sulla cronologia di accesso.

Una volta completati entrambi i passaggi sopra descritti, gli strumenti aggiuntivi saranno installati sul tuo sistema e saranno disponibili per l'uso nel tuo ambiente di sviluppo.

#### 5. Installazione di Oh My Posh

Per installare Oh My Posh utilizzando Winget, esegui il seguente comando in una finestra di PowerShell:

```powershell
winget install JanDeDobbeleer.OhMyPosh -s winget
```

Questo comando utilizzerà Winget per cercare il pacchetto Oh My Posh nell'elenco dei pacchetti disponibili e lo installerà sul tuo sistema. Assicurati di avere una connessione Internet attiva durante l'esecuzione di questo comando in modo che Winget possa scaricare e installare Oh My Posh correttamente.

Una volta completato il processo di installazione, Oh My Posh sarà disponibile sul tuo sistema e potrai iniziare a utilizzarlo per personalizzare il prompt di PowerShell con temi e stili avanzati. Assicurati di configurare Oh My Posh nel tuo file di profilo di PowerShell per attivarlo e sfruttarne appieno le funzionalità.

### Configurazione di PowerShell

1. **Apertura del File di Profilo di PowerShell**:

    Per aprire il file di profilo di PowerShell, esegui il seguente comando in una finestra di PowerShell:

    ```powershell
    notepad $PROFILE
    ```

    > **Nota:** Se il comando precedente restituisce un errore, potrebbe significare che il file di profilo non esiste ancora. In tal caso, crea il profilo eseguendo il seguente comando:

    ```powershell
    New-Item -Path $PROFILE -Type File -Force
    ```

2. **Aggiunta delle Configurazioni al File di Profilo**:

    Una volta aperto il file di profilo di PowerShell, aggiungi le seguenti configurazioni alla fine del file:

    ```powershell
    # Prompt
    oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH/catppuccin_frappe.omp.json" | Invoke-Expression

    # Terminal Icons
    Import-Module Terminal-Icons

    # PSReadLine
    Import-Module PSReadLine
    Set-PSReadLineKeyHandler -Key Tab -Function Complete
    Set-PSReadLineOption -PredictionViewStyle ListView
    ```

    Queste configurazioni personalizzano il prompt di PowerShell utilizzando Oh My Posh con il tema specificato e aggiungono funzionalità avanzate come le icone del terminale e il completamento delle righe di comando.

3. **Applicazione delle Configurazioni**:

    Dopo aver aggiunto le configurazioni al file di profilo, esegui il seguente comando in una finestra di PowerShell per applicare le modifiche:

    ```powershell
    . $PROFILE
    ```

    Questo comando esegue il file di profilo di PowerShell, applicando così le nuove configurazioni al tuo ambiente di sviluppo.

Una volta completati tutti i passaggi sopra descritti, la configurazione di PowerShell sarà completata e potrai godere di un'esperienza più personalizzata e avanzata quando utilizzi PowerShell nel tuo sistema.
